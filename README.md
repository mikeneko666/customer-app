# 顧客管理アプリ

## 概要

JavaScriptを用いて作成した顧客管理アプリです。
社内での顧客情報管理を想定して開発しました。

## 作成背景

業務で使用されるシンプルな顧客管理ツールをイメージし、
データの追加・削除・検索といった基本機能を実装しました。

## 機能

* 顧客情報の追加（名前・電話番号・メモ）
* 顧客情報の削除
* 名前による検索機能（部分一致）
* localStorageによるデータ保存（リロード後も保持）

## 工夫した点

* 配列とオブジェクトで顧客データを管理
* filterメソッドを使用した検索機能の実装
* idを用いたデータ管理で削除処理の正確性を向上

## 使用技術

* HTML
* CSS
* JavaScript
